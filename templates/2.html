Data Types

In Python, variables can store 4 main types of data:
  • String (str): A sequence of characters. Example: "Hello world!"
  • Integer (int): A whole number, positive or negative. Examples: 6, -720, 86400
  • Float (float): Floating-point values, also known as decimals. Examples: 54.9872, -26.8
  • Boolean (bool): A binary data type that can only be either true or false. Examples: True, False
To initialize a variable, just name it and define its value after an equals sign. For example: In this case, name is a string, age is a float, and grade_seven is a Boolean value. You can change a variable in the same way. Remember to refer to your variable by its name. For example: Try adding

print(name)
print(age)
print(grade_seven)

to the above code and pay attention to the output. A Python variable will always take on the last value you assigned to it.

You can also use the function type(var) to find a variable's data type. In the above example, type(name) would output str, type(age) would output float, and type(grade_seven) would output bool.

Finally, you can use the following functions to change a variable's type.
  • str(var) converts the variable to a string;
  • int(var) converts the variable to an integer;
  • float(var) converts the variable to a floating-point value;
  • bool(var) converts the variable to a Boolean value.
